- **Reverse Calculation**: Enter the amount you want to **receive in your account** and instantly know how much to charge the buyer
- Supports **Domestic** and **International** transactions
- Pre-built fee presets (e.g. US Domestic 3.49% + $0.49)

## 📋 How to Use

1. Choose calculation mode:
   - **Charge Amount** (how much the buyer pays)
   - **Desired Net Amount** (what you want to actually receive)

2. Enter the transaction amount

3. Select or customize the fee rate (percentage + fixed fee)

4. Choose currency and transaction type (Domestic / International)

5. View instant results:
   - PayPal Fee
   - Net Amount Received (or Recommended Charge Amount)
